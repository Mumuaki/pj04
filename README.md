# Электронная сервисная книжка "Мой Силант"

Django-приложение для управления сервисным обслуживанием техники Силант.

## Возможности

- **Управление машинами**: полная информация о технике (модели, серийные номера, комплектация)
- **Техническое обслуживание**: учет проведенных ТО с датами и наработкой
- **Рекламации**: регистрация отказов и способов восстановления
- **Роли пользователей**: Клиент, Сервисная организация, Менеджер (продавец своей компании)
- **Поиск по заводскому номеру**: для неавторизованных пользователей
- **Адаптивный дизайн**: работает на всех устройствах

## Технологии

- **Backend**: Django 4.2.27
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **База данных**: SQLite (разработка), PostgreSQL (продакшн)
- **Аутентификация**: Django-allauth
- **API**: Django REST Framework

## Установка и запуск

### Требования

- Python 3.8+
- pip
- virtualenv (рекомендуется)

### Шаги установки

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/Mumuaki/pj04.git
   cd MySilant
   ```

2. **Создать виртуальное окружение**
   ```bash
   python -m venv venv
   ```

3. **Активировать виртуальное окружение**
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   - Linux/Mac:
     ```bash
     source venv/bin/activate
     ```

4. **Установить зависимости**
   ```bash
   pip install -r requirements.txt
   ```

5. **Применить миграции**
   ```bash
   python manage.py migrate
   ```

6. **Загрузить начальные данные**
   ```bash
   python manage.py loaddata fixtures/initial_data.json
   ```

7. **Запустить сервер разработки**
   ```bash
   python manage.py runserver
   ```

8. **Открыть в браузере**
   ```
   http://127.0.0.1:8000/
   ```

### Дополнительно

Для продакшена выполните сбор статических файлов:
```bash
python manage.py collectstatic
```
